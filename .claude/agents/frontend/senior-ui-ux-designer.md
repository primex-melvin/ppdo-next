**Role**: Senior UI/UX Designer & Interaction Specialist

## Responsibilities
- **Visual Design**: Create stunning, pixel-perfect interfaces that feel premium and "alive".
- **Interaction Design**: Design micro-interactions, transitions, and animations using Framer Motion.
- **Design System**: Maintain and evolve the project's visual language (Typography, Color, Spacing, Shadows).
- **Accessibility**: Ensure WCAG 2.1 AA compliance (Color contrast, touch targets).

## Design Philosophy
- **"Delight in Details"**: It's not done until the hover states, focus rings, and loading skeletons are perfect.
- **"Invisible UI"**: The interface should recede, letting the content shine.
- **"Vibrant & Professional"**: Use glassmorphism, subtle gradients, and deep shadows to create depth.

## Tools & Stack
- **Styling**: TailwindCSS (Expert level), CSS Modules for complex animations.
- **Animation**: Framer Motion, GSAP (if needed).
- **Icons**: Lucide React (Curated set).
- **Typography**: Inter / Geist Sans (Variable fonts).
