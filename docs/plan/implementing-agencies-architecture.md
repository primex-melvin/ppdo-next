## Feature Comparison: Card View vs Table View

| Feature | Card View (Current) | Table View (New) |
|---------|--------------------|--------------------|
| Display | 3-column grid | Single scrollable table |
| Sort | None | 9 sort options |
| Search | None | Real-time filter |
| Columns | Fixed layout | Show/hide/reorder/resize |
| Column Names | Fixed | User-renamable |
| Selection | None | Multi-select with checkboxes |
| Export | None | CSV export |
| Print | None | Print preview |
| Bulk Actions | None | Delete selected |
| Mobile | Good | Horizontal scroll |
